Projeto de análise de incidentes operacionais e tecnológicos em ambiente bancário digital, com foco em monitoramento de operações, SLA, MTTR e gestão de incidentes.
Os dados foram gerados de forma sintética em Python, respeitando regras reais de negócio, incluindo duplicidades, histórico de status e diferentes níveis de prioridade. A modelagem foi realizada em SQL no Databricks e os indicadores foram disponibilizados em dashboards no Power BI, simulando um cenário real de Command Center e Operações.
